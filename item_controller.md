# Контроллер услуг

## Создать новую услугу

- **URL:** `/items`
- **Метод:** `POST`
- **Описание:** Создает новую услугу.
- **Тело запроса:** JSON объект с полями `title`, `description` и `gifLink`.
- **Возвращаемый тип:** JSON объект новой услуги.

## Получить услугу по ID

- **URL:** `/items/{id}`
- **Метод:** `GET`
- **Параметры пути:** `id` (ID услуги).
- **Описание:** Возвращает услугу по указанному ID.
- **Возвращаемый тип:** JSON объект услуги.

## Удалить услугу по ID

- **URL:** `/items/{id}`
- **Метод:** `DELETE`
- **Параметры пути:** `id` (ID услуги).
- **Описание:** Удаляет услугу по указанному ID.

## Добавить услугу в категорию

- **URL:** `/items/{itemId}/categories/add/{categoryId}`
- **Метод:** `PUT`
- **Параметры пути:** `itemId` (ID услуги), `categoryId` (ID категории).
- **Описание:** Добавляет услугу в указанную категорию.

## Удалить услугу из категории

- **URL:** `/items/{itemId}/categories/remove/{categoryId}`
- **Метод:** `PUT`
- **Параметры пути:** `itemId` (ID услуги), `categoryId` (ID категории).
- **Описание:** Удаляет услугу из указанной категории.

## Получить список всех услуг

- **URL:** `/items`
- **Метод:** `GET`
- **Параметры запроса:** `page` (номер страницы).
- **Описание:** Возвращает список всех услуг.
- **Возвращаемый тип:** Страница со списком услуг.

## Поиск услуг по названию

- **URL:** `/items/search`
- **Метод:** `GET`
- **Параметры запроса:** `title` (название услуги), `page` (номер страницы).
- **Описание:** Ищет услуги по названию.
- **Возвращаемый тип:** Страница со списком найденных услуг.

## Поиск услуг без категории

- **URL:** `/items/search/withoutCategory`
- **Метод:** `GET`
- **Параметры запроса:** `page` (номер страницы).
- **Описание:** Ищет услуги, не принадлежащие ни одной категории.
- **Возвращаемый тип:** Страница со списком услуг.

## Поиск услуг по категории и названию

- **URL:** `/items/search/byCategoryAndTitle`
- **Метод:** `GET`
- **Параметры запроса:** `categoryId` (ID категории), `title` (название услуги), `page` (номер страницы).
- **Описание:** Ищет услуги по заданной категории и названию.
- **Возвращаемый тип:** Страница со списком найденных услуг.

## Обновить описание услуги

- **URL:** `/items/{id}/description`
- **Метод:** `PUT`
- **Параметры пути:** `id` (ID услуги).
- **Тело запроса:** Новое описание услуги.
- **Описание:** Обновляет описание услуги.

## Обновить ссылку на GIF изображение услуги

- **URL:** `/items/{id}/gif`
- **Метод:** `PUT`
- **Параметры пути:** `id` (ID услуги).
- **Тело запроса:** Новая ссылка на GIF изображение услуги.
- **Описание:** Обновляет ссылку на GIF изображение услуги.
