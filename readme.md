## IVRstand

### Описание:
IVRstand - это приложение для управления услугами и категориями на IVR стенде. Приложение подходит для создания, редактирования, удаления и поиска как услуг, так и категорий, а также создаёт связь между ними (добавление услуги в категорию, удаление из неё).

### Функциональность:
1. Создание, редактирование и удаление категорий услуг.
2. Создание, редактирование и удаление услуг, возможность добавить и удалить из категорий.
3. Поиск категорий по заголовку с поддержкой пагинации.
4. Поиск услуг по заголовку с поддержкой пагинации.
5. Поиск услуг без категории с поддержкой пагинации.
6. Обновление описания и ссылки на GIF-анимацию для услуг.

### Технологии:
1. Java
2. Spring Framework (Spring Boot, Spring Data JPA)
3. Hibernate (для работы с базой данных)
4. Lombok (для уменьшения количества шаблонного кода)
5. RESTful API
6. Git (для управления версиями)

### Структура проекта:
- `com.good.ivrstand.app`: содержит классы сервисного слоя, отвечающие за бизнес-логику приложения.
- `com.good.ivrstand.domain`: содержит классы сущностей (Entity) для описания объектов в базе данных.
- `com.good.ivrstand.extern`: содержит модели и контроллеры для связи с фронтендом.
- `resources`: содержит файлы конфигурации и SQL-скрипты для базы данных.
- `test`: содержит модульные тесты для проверки функциональности приложения.
