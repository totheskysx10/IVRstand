# Контроллер категорий

## Создать новую категорию

- **URL:** `/categories`
- **Метод:** `POST`
- **Описание:** Создает новую категорию услуг.
- **Тело запроса:** JSON объект с полем `title`.
- **Возвращаемый тип:** JSON объект новой категории.

## Получить категорию по ID

- **URL:** `/categories/{id}`
- **Метод:** `GET`
- **Параметры пути:** `id` (ID категории).
- **Описание:** Возвращает категорию по указанному ID.
- **Возвращаемый тип:** JSON объект категории.

## Удалить категорию по ID

- **URL:** `/categories/{id}`
- **Метод:** `DELETE`
- **Параметры пути:** `id` (ID категории).
- **Описание:** Удаляет категорию по указанному ID.

## Получить список всех категорий

- **URL:** `/categories`
- **Метод:** `GET`
- **Параметры запроса:** `page` (номер страницы).
- **Описание:** Возвращает список всех категорий.
- **Возвращаемый тип:** Страница со списком категорий.

## Добавить подкатегорию к категории

- **URL:** `/categories/{categoryId}/parent/set/{parentId}`
- **Метод:** `PUT`
- **Параметры пути:** `categoryId` (ID подкатегории), `parentId` (ID будущего родителя).
- **Описание:** Добавляет подкатегорию к указанной категории.

## Удалить подкатегорию из категории

- **URL:** `/categories/{categoryId}/children/remove/{childId}`
- **Метод:** `PUT`
- **Параметры пути:** `categoryId` (ID родителя), `childId` (ID удаляемой подкатегории).
- **Описание:** Удаляет подкатегорию из указанной категории.
